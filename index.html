Summary

In Project Two, we scraped NBA data from NBA.com with objectives to analyze various metrics and find the optimal fantasy basketball lineup. Unfortunately, the steep learning curve of Flask and JavaScript left us with little time to analyze the data and make fantasy basketball predictions. For this final project, we revisited the data scraping tools from Project Two and added machine learning and Tableau visualizations to take a deeper dive into the data and fulfill our objective from Project Two.


FanDuel Point Structure

Blocks (B): 3 Points
Field Goal Made (FG): 2 Points
Free Throw Made (FT): 1 Points
Rebounds (R): 1.2 Points
Steals (S): 3 Points
Turnovers (T): -1 Point
